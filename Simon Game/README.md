#  Simon Game Javascript

A browser-based Simon Memory Game built using HTML, CSS, and JavaScript.  
The game generates a random sequence of colors that the player must repeat correctly to advance to the next level.

---

##  Features

- Dynamic random sequence generation
- Level progression system
- Real-time user input validation
- Game over detection and restart functionality
- Interactive button flash animations

---

##  Technologies Used

- HTML5
- CSS3 (Grid Layout & Animations)
- JavaScript (DOM Manipulation & Event Handling)

---

##  How to Play

1. Press any key to start the game.
2. Watch the button sequence carefully.
3. Repeat the sequence by clicking the buttons in the same order.
4. Each level adds a new button to the sequence.
5. The game ends if you click the wrong button.
6. Press any key to restart after Game Over.
